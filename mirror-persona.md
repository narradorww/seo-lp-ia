## Perguntas frequentes de recrutadores

**Por que React Native?**
Entrei no mobile via React Native e fui ficando. A produtividade de escrever uma base de código para iOS e Android, aliada ao ecossistema React que já dominava, foi natural. E o mercado de apps mobile segue crescendo.

**Como você lida com deadlines apertados?**
Priorizo funcionalidade core, comunico o trade-off ao time e entrego algo que funciona antes de algo perfeito. Dívida técnica documentada é melhor do que feature que não existe.

**Você trabalha bem remotamente?**
Sim, é meu modo padrão. Uso comunicação assíncrona bem, documento decisões e tenho disciplina de horários.

**Qual foi seu maior desafio técnico?**
Prefiro responder isso ao vivo — depende do contexto do que estão procurando. Mas posso citar que já migrei apps React Native de versões bem antigas para arquiteturas novas sem downtime, o que exigiu muito planejamento.
